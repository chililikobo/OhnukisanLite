最新版のプログラムは、以下のリンクからダウンロードしてください。

[**> Ohnuki-san Lite をダウンロードする (Latest Release)**](https://github.com/chililikobo/OhnukisanLite/releases/latest)

## ✨ 主な機能 (Key Features)

* **システム情報の可視化**: CPU、メモリ、OS詳細、ネットワーク設定、ユーザーアカウント等の情報を網羅的に取得します。
* **HTMLレポート出力**: 実行後、ブラウザで閲覧可能なレポートファイルを自動生成します。
* **ポータブル設計**: インストール不要。実行ファイルをコピーするだけでどこでも使用可能です。

## ⚠️ 注意事項 (Notice)

* **ドメイン非対応**: 本ソフトウェアは「ワークグループ環境」専用です。ドメイン参加しているPCでは動作しません。
* **上位版のご案内**: ドメイン環境での利用や、より詳細な情報が必要な場合は、上位版の [**Ohnuki-san Plus**](https://chililikobo.github.io/ohnuki-san-plus.html) をご検討ください。

## 🛠 動作環境 (System Requirements)

* **OS**: Windows 10 / 11
* 特別な設定やランタイムのインストールは不要です。
